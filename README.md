# Portafolio Personal | Lina Marcela Rodríguez Rincón

Portafolio web interactivo desarrollado con **HTML, CSS y JavaScript vanilla**. 
Este proyecto muestra mi perfil profesional como Tecnóloga en Desarrollo de 
Software y mi experiencia en el sector.

## Características

- **Modo Oscuro/Claro** con preferencia en `localStorage`
- **Diseño responsive** adaptable a móviles, tablets y desktop
- **Interfaz limpia y moderna** con variables CSS personalizadas
- **Botón flotante** para cambiar el tema en cualquier momento
- **Enlaces directos** a LinkedIn y GitHub
